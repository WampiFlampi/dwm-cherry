## Dependencies:
###### xorg/xlibs 
   1. ```pacman -S xorg-server...```

   2. ```apt-get install xorg-server...```

###### compton/xcompmgr

   1. ```pacman -S compton```

   2. ```apt-get install compton```

Just use default config from their github
I patched st already
    
    
## Building:

  1. ```cd dwm```
   
  2. ```sudo make clean install```
   
  > same process for st
